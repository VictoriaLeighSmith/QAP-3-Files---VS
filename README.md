2. Web Dev - Mo's Lawncare Invoice Generator

**QAP 3 Python - Description:

This Python program calculates the final purchase or payment price of a used vehicle based on user inputs. It includes functions for formatting, validating inputs, and generating a summary of the customer’s purchase information.

How to run:

1. Ensure all files are located in one folder
2. Run program using:

   python3 HonestHarry.py
3. Follow on-screen prompts to enter vehicle and customer information.

**QAP 3 Web Dev - Description:

A JavaScript-based web page that calculates and displays a formatted invoice for Mo's Lawncare Services. The program prompts the user for customer details and property size, then applies the business rules for border trimming, mowing, fertilizer, and taxes.

How to run:

1. Open QAP 3 - Web folder
2. Open 'index.html' in any browser.
3. Follow the prompts to view the generated invoice table.
